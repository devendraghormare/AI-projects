## âš™ï¸ Tech Stack

* **Backend** : FastAPI, SQLAlchemy (async)
* **LLM** : OpenAI GPT-4o / GPT-3.5

* **Container** : Docker + Docker Compose
* **Extras** : Loguru, Prometheus (optional), Pydantic, Uvicorn
